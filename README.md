# HyojaBall

This project is a website for visualizing and analyzing the data of KBO players.

## Setup

### set .env files

- .env 파일 setting/dev 위치에 .env 파일 생성.

## Usage

- 도커 처음 시작시.

```shell
 $ docker-compose up --build
```

- 빌드 후 compose up 만 실행.

```shell
 $ docker-compose up
 $ docker-compose down # 프로젝트 마친 후 실행
```
